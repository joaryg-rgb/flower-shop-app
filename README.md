# Flower Shop Web Application

Simple web application that allows users to calculate the total price of a flower order.
The application uses an HTML form to collect user input and PHP to process the data and calculate the total cost of the order.  
Users can select different types of flowers and specify quantities, and the application calculates the final price.

## Features
- Multiple flower selection
- Quantity input
- Automatic total price calculation
- Simple and intuitive interface with images
